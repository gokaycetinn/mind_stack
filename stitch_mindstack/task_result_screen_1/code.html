Why this works

By defining the base case first if (n <= 1) return 1, you prevent the infinite loop. The recursive step then correctly breaks the problem down. 

function fact(n) {
  // Correct logic here
  if (n <= 1) return 1;
  return n * fact(n - 1);
}
